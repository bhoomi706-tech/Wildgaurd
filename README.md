# 🌿 WildGuard IoT System

## 🚀 Overview
WildGuard is an IoT-based smart wildlife monitoring and protection system designed to prevent human-animal conflicts and ensure real-time tracking and surveillance.

---

## 🧠 Features
- 📍 GPS Tracking for location monitoring
- 🎥 ESP32-CAM for live video feed
- 🔴 IR Sensors for obstacle/animal detection
- 📡 RFID for identification
- 🚗 Motor Control using TB6612 Driver
- ⚡ Efficient Power Management (12V → 5V → 3.3V)

---

## 🧩 Components Used
- ESP32
- ESP32-CAM
- Neo-6M GPS Module
- RFID RC522
- IR Sensors
- TB6612 Motor Driver
- 12V DC Motors
- Buck Converter (LM2596)
- AMS1117 Voltage Regulator

---

## 🔌 System Architecture
- 12V supply powers motors
- Buck converter steps down to 5V
- AMS1117 regulates 3.3V for ESP32 and sensors
- ESP32 acts as the main controller
